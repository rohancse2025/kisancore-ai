# IoT

Placeholder for device code and message formats.

Suggested structure:
- `firmware/` (ESP32/Arduino/etc.)
- `simulator/`
- `schemas/` (JSON schemas for telemetry)

ESP32 GPIO26 -> Relay IN pin
Relay VCC -> 5V
Relay GND -> GND
Relay NO -> Fan/Pump positive
Fan/Pump negative -> Power supply negative
